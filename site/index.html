So, why Linux?

Who is Linus Torvalds?

The main Linux kernel developer since 1991, still active maintainer. Developed 'Git', the open-source control management system.

Fun fact:

If someone tells you they run on Linux, they are either lying, not informed enough or thinking you wouldn't understand (which is totally fine if you are not into STEM)-- Linux is the kernel of an operating system, people's hardware runs on OSs based on that kernel -- Ubuntu, Debian, Manjaro, Elementary, Linux Mint, Arch, Fedora, Kali Linux and many more... The most used ones ar probably Ubuntu and Linux Mint. As Arch is mainly used from a console its considered the "hard-core" distro.
